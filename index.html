Permutations: minimum reachable p ≈ 1/(perms+1) — 1000→0.001, 5000→0.0002, 10000→0.0001. For publication-grade tiny p.adj, use ≥5000. Seed: set an integer (e.g. 42) for reproducible runs; clear it for fully random permutations. Gene collapse: when the same gene appears multiple times in DEP (isoforms, multiple sites), only the row with largest |metric| is kept for ranking. Collapsed genes are tagged in the leading edge as GENE(×N) where N is the number of original rows.
